# Installation Instructions

## Prerequisites
- Ensure you have the latest version of Python installed.
- Install any necessary dependencies listed in `requirements.txt`.

## Steps to Install
1. Clone the repository:
   ```bash
   git clone https://github.com/Juju-62/FS25_WheelSlipDisplay.git
   ```
2. Navigate into the directory:
   ```bash
   cd FS25_WheelSlipDisplay
   ```
3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:
   ```bash
   python main.py
   ```

For additional information, please refer to the official documentation or contact the maintainers.
